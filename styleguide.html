Gadget styles example usage

This repository defines some common styles for Versal gadgets. Such gadgets are typically instantiated inside an iframe, so it is the developer's responsibility to include styles that makes the gadget look consistent with other gadgets. This repo helps with that.

This style guide is still in its infancy, we will update it along with the CSS to show how to use it.

Note that this style guide is probably going to be unorthodox. Gadgets are unlike regular pages on the web, they typically contain no pre-set text, except for some labels. This is because they are meant to be customized by authors of Versal courses. Therefore, you will not find regular typographic components as seen in most style guide, but mostly elements for making dynamic interfaces.
